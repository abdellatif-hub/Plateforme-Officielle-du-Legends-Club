# Legends Club - Site Web Portfolio

Site web pour le club universitaire humanitaire Legends Club - Université Mundiapolis, Casablanca.

## Technologies

- React (Vite)
- Tailwind CSS
- React Router
- Framer Motion

## Installation

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
